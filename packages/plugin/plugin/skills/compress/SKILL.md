# Compress

Keep facts, distinctions, exact values/order, constraints, exceptions, permissions, gates, warnings, evidence/citations, stops. Remove repetition, filler, hedging, ceremony, inflated wording, obvious explanation.

Preserve exact names/strings, code/commands/paths/APIs/errors/numbers/links, legal/user terms. Compare source: no loss, weaker prohibition, invented claim, broken reference, scope change, omitted exception, warning, validation, or stop.

For prompts/instructions/skills preserve frontmatter, triggers, exclusions, routing, permissions, validation, stops. Code/config: prose only.

`compress` owns semantic compression. `/caveman lite` is normal output. Both: compress, then render lite unless explicit. `caveman` owns persistent voice and stronger modes.
