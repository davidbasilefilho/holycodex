# UI/UX data

CSV files are lookup tables, not mandatory style. Query only needed domain: colors, typography, landing structure, charts, icons, UX guidance, product/style examples, framework stack, or React performance. Filter by project context and accessibility; established design contract wins. Never copy labels/content blindly.

Use `python scripts/search.py <query>` for narrow search or `python scripts/design_system.py` when a requested greenfield design system needs combined suggestions.
